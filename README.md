# RPI NSBE — Financial Literacy Tool

An interactive financial planning web app built for RPI's NSBE chapter to help engineering students visualize the long-term impact of investing early. Students select their major, see their expected starting salary from RPI Class of 2025 data, set an investment strategy, and watch a compound growth chart project their portfolio to retirement.

Built for in-person presentations and designed to work equally well on a projector or a student's phone.

---

## Features

- **Major selector** — searchable, filterable card grid of RPI majors with average starting salaries sourced from the RPI Class of 2025 CCPD report, including salary range visualization
- **Grad school estimator** — optional collapsed panel that applies historical Master's (15–20%) and PhD (20–30%) salary multipliers and lets students use the projected midpoint as their salary input
- **Hourly wage converter** — optional panel that converts an internship hourly rate to an annual salary and populates the salary field
- **Investment inputs** — editable annual salary, monthly investment percentage slider (1–30%), starting age stepper, and a three-scenario market return selector (Conservative 7%, Average 11%, Optimistic 15%)
- **Compound growth chart** — three-line area chart (Recharts) projecting portfolio growth from starting age to 65 across all three return scenarios simultaneously, with end-of-line value badges, a hover tooltip, and a Y-axis domain scaled to the data
- **Impact callout** — dynamic sentence beneath the chart contextualizing the result with multiplier, total contributions, and interest earned
- **Fully mobile-responsive** — designed for 375px phones through desktop projectors, with touch-friendly tap targets and a compact step tracker on small screens

## Getting started

**Prerequisites:** Node 18 or later.

```bash
# Clone the repo
git clone https://github.com/SoloStarkes/NSBE-RPI_FIRE.git
cd NSBE-RPI_FIRE/finance-viz

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open `http://localhost:5173`.

```bash
# Production build
npm run build

# Preview the production build locally
npm run preview
```

---

## Deployment

The app is deployed on Vercel. Any push to `main` triggers an automatic redeploy.

**Manual deploy setup:**
1. Import the repo into Vercel
2. Set **Root Directory** to `finance-viz`
3. Framework preset auto-detects as Vite; output directory is `dist`
4. No environment variables required

## Data source

Starting salary data is sourced from the **RPI Office of Career and Professional Development (CCPD) Class of 2025 First Destination Survey**. All figures are self-reported averages and are used for educational illustration only.

---

## Disclaimer

Projections generated by this tool are illustrative estimates based on monthly compound interest with consistent contributions and no interruptions. They do not account for taxes, inflation, management fees, or market volatility. Past market returns do not guarantee future results. This tool is for educational purposes only and does not constitute financial advice.
